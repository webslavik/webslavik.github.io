Обо мне

Мои навыки

Стек того, что использую при работе с небольшим описанием/мнением:

HTML/Jade
Jade расширяет возможности написания HTML переменными, циклами, функциями и тд. Основной шаблонизатор, который я использую.
CSS/Sass
Sass является главным препроцессором для написания стилей. Уже тяжело представить свою работу без него.
JavaScript
К большой радости написание на чистом JS с каждой редакцией языка становиться все проще. Правда использование jQuery пока никуда не уходит.
jQuery
Самая популярная библиотека для написания JavaScript которую использую и я.
Angular
Замечательный JS фреймворк от Google.
Bootstrap
Раньше часто использовался мною в работе, но с увеличением опыта понял, что он накладывает определенные ограничения.
Foundation for Emails
Фреймворк который я использую для написания писем. Хорошая документация и автоматизация процесса делают его незаменимым.
Gulp
Без этого таск-менеджера тяжело представить современный фронтэнд. Берет на себя многие рутинные дела.
Photoshop/Illustrator
Не скажу что мне нравиться Photoshop, слишком большой и неповоротливый. Надеюсь с выходом Adobe XD многое поменяется. Illustrator же отлично подходит для вектора.
Bower/npm
Умение пользоваться этими пакетными менеджерами необходимо. Правда сейчас npm вполне заменяет Bower.
Git
Система контроля версий используемая мной в каждом проекте.
Как правило проекты я храню на Github и Bitbucket.
Jekyll
Это генератор статических сайтов который я использовал для создание своего сайта. В отличие от CMS(WordPress, Modx и др.), Jekyll легок и быстр, его можно развернуть на Github Pages.
