Обо мне

Мои навыки

HTML/Jade
Этот шаблонизатор великолепен! Разбить один большой файл на мелкие кусочки, ввести переменные, написать функции, использовать циклы и еще много другого. То, чего не хватало в написании разметки.
CSS/Sass
Sass это уже норма. Это - “по умолчанию”. Это целый завод по написанию CSS, который вам даже не понадобиться и скорее всего так и будет.
JavaScript/jQuery
Тяжело представить себе веб без JS. Поэтому знать JS надо.
Еще надо знать jQuery.
Angular
Пока могу сказать, что изучение этого фреймворка очень интересный опыт. Еще все впереди.
Bootstrap
Знать Bootstrap необходимо. Использовать? Это уже каждый решает сам. Не всегда и везде он подойдет, но все зависит от случая.
Foundation for Emails
Написание писем это ад. Выбраться из него помогает этот фреймворк. Легок в освоении, есть хорошая документация плюс полная автоматизация процесса. Прелесть.
Gulp
Не представляю своего рабочего процесса без этого таск-менеджера. Он поможет выполнить всю ту рутину, которая на ваших плечах. Минификация, локальный сервер, создание спрайтов и многое другое. Каждый фронтэндер должен его использовать.
Photoshop/Illustrator
Оба редактора используются мной в работе. Первый часто, второй пореже. Для создания макетов можно было бы пользоваться и чем то другим (Sketch only Mac), но с головой хватает и Photoshop.
Bower/npm
Умение пользоваться этими пакетными менеджерами - повышает количества вашего свободного времени и добавляет +1 к скиллу.
Git
Система контроля версий используемая мной в каждом проекте.
Как правило проекты я храню на Github и Bitbucket.
Jekyll
Это генератор статических сайтов который я использовал для создание своего сайта. В отличие от CMS(WordPress, Modx и др.), Jekyll легок и быстр, его можно развернуть на Github Pages.
